生成式人工智慧 (Generative AI，GenAI 或是 GAI) 源於深度學習 (Deep Learning，DL)。深度學習加強了機器辨識的能力，進而成為圖形辨識或是自然語言處理等領域的核心。生成式人工智慧與深度學習雖然同源，但是與深度學習致力於分類與識別資料，生成式人工智慧專注於分析資訊後，進而從原本的資料創造出類似的資料。

雖然目的不同，但是生成式人工智慧與深度學習底層的差異卻很小，主要核心依舊是：

- 識別資訊 - 藉由識別輸入，預測產出
- 資料分類 - 找出與輸入相近的資訊類別
- 神經網路 - 藉由輸入與輸出類別的關係，提高生成資訊的準確率

目前生成式人工智慧目前的方向

- 生成對抗網路 (Generative adversarial networks, GAN)

	非監督式學習的一種方式。透過兩個神經網路的對抗，一個負責生產一個負責檢驗，直到檢驗方無法識別生產方給予的資訊是否為生產方創造
	
- 變動自編碼機 (Variational autoencoders，VAE)
- 擴散模型 (Diffusion models)

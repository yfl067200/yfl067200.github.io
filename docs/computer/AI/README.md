人工智慧 (Artifical Intellenge，AI) 最初是一種嚴格定義的統計模型，用來幫助我們分析與理解資訊。經歷了多年的發展，結合了計算機與統計模型，進而發展出機械學習 (Machine Learning，ML)；藉由機械學習，我們可以發現資料之間的複雜關係，進而改善統計模型。

機械學習之後，引導出深度學習 (Deep Learning，DL) 與其核心基礎的神經網路 (Neural Networks)。深度學習提升了辨識的能力，進而在圖形辨識、聲音辨識、與自然語言處理等領域成為核心工具。

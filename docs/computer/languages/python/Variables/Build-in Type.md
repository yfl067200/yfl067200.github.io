
以下為 Python 內建支援的資料型態

| Type           | Description                                                                                                                                                                                                                                                              |
| :------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| None           | 單一物件內涵一個 Value<p>可以透過 Python 的保留字 `None` 存取該變數，即 Python 內建一個該型                                                                                                                                                                                                           |
| NonImplemented | 單一物件內涵一個 Value<p>可以透過 Python 的保留字 `NonImplemented` 存取該變數，即 Python 內建一個該型態的變數<p>當無法 link 到函式時，Python 將丟出改變數<p>==無法==使用布林值比                                                                                                                                                |
| Ellipsis       | 單一物件內涵一個 Value<p>可以透過 Python 的保留字 `Ellipsis` 或是 `.                                                                                                                                                                                                                       |
| numbers.Num    |                                                                                                                                                                                                                                                                          |
| Sequences      | 表示一個有限且有序的資料集合，資料的 Index 為不重複的正整數<p>需要實作 len() 函式，回覆資料集合的總數<p>透過 Index 存取資料時，若是無法取得資料，將產生一個 IndexError 的例外<p>另一個需要的是支援切割 (slice) 功能，可以透過 Sequences[START:STOP[:skip count]] 截取子集合<p>Immutable Sequences 包含 string、tuple、與 bytes<p>Mutable Sequences 包含 list、與 byte array |
| Sets           | 一個有限，但是無序，且無重複的 Immutable 資料集合<p>無法使用 Index，也不支援 slice                                                                                                                                                                                                                   |
|                |                                                                                                                                                                                                                                                                          |

## 數學運算元

| 運算元      | 說明                                                                |
| :------- | :---------------------------------------------------------------- |
| x + y    | 加法                                                                |
| x - y    | 減法                                                                |
| x \* y   | 乘法                                                                |
| x \*\* y | 次方，或稱為冪。其值為 $x ^ y$                                               |
| x \/ y   | 除法 <p>用於 Python 2 時，只要被除數或是除數為==浮點數==，其結果等同於此運算元                  |
| x \/\/ y | 四捨五入的除法<br>Python 2 並未支援此願算元；當被除數與除數皆為==整數==時，其結果為整數，<br>但是不會四捨五入 |

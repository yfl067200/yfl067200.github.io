
Lua 跟 Python 都是解釋型語言 (interpreted language)；他不需要先經過編譯器轉換成機器碼，反之是透過直譯器 (interpreter) 一行行將程式碼進行解釋並直接執行。另一項兩者都一樣的部分，是這兩種語言都能與 C 語言合作


> [!Note]
> 在 hello.lua 檔案中寫入程式碼 `print("Hello World!")`，即可透過 Lua 直譯器執行
> prompt>  lua hello.lua
> Hello World!

Lua 設計的初衷在於提供使用者一個比較友好的開發環境，讓使用者專注於邏輯而非需要將注意力放在資料結構的細節上

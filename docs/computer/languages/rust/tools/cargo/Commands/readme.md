
命令格式：

- cargo [options] command [args]
# Options

## Special Options

| Options          | 縮寫  | 說明                                                  |
| :--------------- | --- | --------------------------------------------------- |
| --color $WHEN    |     | cargo 輸出字串是否著色<br>WHEN 包含 3 個選項：auto、always、與 never |
| --explain <code> |     |                                                     |
| --list           |     |                                                     |
| --version        | -V  |                                                     |
| --verbose        | -v  | 提供更多說明，-vv 將提供最詳細的資訊                                |

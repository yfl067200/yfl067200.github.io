透過變動 ${HOME}/.vimrc 檔案內容，進行 vim 客製化。

不同版本的 vim 支援的客製化設定不同。建議在改寫 .vimrc 檔案之前，可以先在 vim 環境中，透過命令 `:set` 測試客製化命令；或是透過 `:help` 命令，查看客製化命令的說明。以下為執行 `:help tabstop` 命令的結果

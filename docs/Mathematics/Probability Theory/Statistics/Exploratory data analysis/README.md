探索性資料分析 (Exploratory data analysis，EDA) 的目的在於理解資料。資料的出處不同，其內容的格式也不盡相同；統計學的其中一大難題在於如何將一系列雜亂無章的資料，轉化為結構化資料，以便進行下一階段的行動。

# 結構化資料 (structured data)

結構化的資料可以粗分為兩類：
1. 數字化的資料 (numeric)
    - 連續性資料 (continuous)：一段時間之內，依照相同間隔時間進行的採樣。
    - 離散性資料 (discrete)
2. 有關聯性的資料 (categorial)
    - 無序的有限資料集合
    - 二元資料 (binary)：當資料只能分為兩類
    - 有序 (Ordinal)

> [!NOTE]
> 為了方便進行資料分析與分類，通常資料都會轉化為數字；即便是有關聯性的資料，其內部元素大多數也用數字。我們用一串文字舉例，在自然語言處理 (Natural Language Processing，NLP) 會將每個重要字轉化成獨一無二的數字。

## 位置評估 (estimates of location)

將資料數字化後，可能會遇上的一個問題就是，每一個數值之間的計離過大或是過小。為了方便處理，會將臨近的數值改為同一個值，姑且先稱為評估值 ($\bar{x}$)；為了能反應出該區塊資料的代表性，常用的求值的方式有：

1. 平均值 (mean)
2. 中位數 (median)

雖然平均數是最容易被採用的方法，但是有極端值 (outliers) 的情況下平均數反倒無法反映出正確的事實。雖然可以透過加權平均值 (weighted mean) 或是修剪平均值 (trimmed mean) 來減少或是移除極端值的影響，但是改用中位數可能更能反映事實。

> [!NOTE]
> 中位數也可能因為極端值而失真；當資料出現的頻率差距過大，加權中位數比較能反映現實。加權中位數的流程為紀錄各數字出現的頻率，將頻率加總之後除以 2，即可得到加權後的中點。再從原本的資料找出位於加權後中點的值，即為加權中位數。

## 離散程度評估 (estimates of variability)

透過位置評估後，可以獲得一個代表某一類別的值；現在要來驗證該值是否能代表該類別。俗話說物以類聚，人以群分；同一類的資料其差異應該不大。因此透過離散程度頻估，可以確認位置頻估得到的值是否有代表性；亦可用來作為挑出極端值或是雜訊的工具。



### 標準差與相關評估方式

最常使用的離散程度頻估就是使用差距。下表為相關名詞的解釋

| 名詞                                 | 說明                                                                                |
| :--------------------------------- | :-------------------------------------------------------------------------------- |
| 誤差 (Deviations)                    | 資料與評估值之間的差距<br>因為誤差有正負，直接加總會導致資料失真                                                |
| 平均差 (Mean Absolute Deviations，MAD) | 將誤差的絕對值加總之後，求平均值<br>公式如右 ${\sum^n_i \|x_i - \bar{x}\|}\over n$，其中 n 為樣本數          |
| 方差 (Variance)                      | 各誤差平方的總和，再除以 n - 1<br>公式如右 ${\sum^n_i (x_i - \bar{x})^2} \over {n - 1}$，其中 n 為樣本數 |
| 標準差 (Standard deviation)           | 方差的平分根<br>公式如右 $\sqrt{{\sum^n_i (x_i - \bar{x})^2} \over {n - 1}}$，其中 n 為樣本數      |

## 矩形數據 (rectangular data)

結構化資料的最後一步，就是將資料製作成矩形數據；多是二維矩陣 (two-dimensional matrix)，每一行代表一個紀錄，每一列代表一個變量/特徵。


理解的過程會經歷以下流程：

1. 構思關於這筆資料的問題。
2. 透過視覺化、轉換和建模，試著尋找答案。
3. 經由所得的結果，重整原先的問題，或者構思、增加新的問題。
